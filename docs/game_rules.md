# CARDS ON THE GRID（仮）仕様書 v0.3

## ゲーム概要

ターン制で、盤面上のモンスター移動とカード使用を組み合わせたボード×カードゲーム。

- デッキ構築要素なし（共通山札方式）
- 思考スピードを要求しない
- 非公開情報は「手札」「補充」「地雷」
- テンポ重視
- カード1枚で読み合いが発生する設計
- 不利側が逆転できる要素を残す（地雷など）

## 用語

- *You*：自分
- *Opponent*：対戦相手
- *Players*：*You* と *Opponent* の総称
- *Creatures*：盤面上に配置されるコマ
- *Successors*：補充フェイズで新たに召喚される *Creature*
- *Successor cost*：*Creature* が死亡してから、次の *Successor* が補充されるまでに必要なターン数

## 盤面仕様

- 盤面サイズ：7×7
- 各マスは正方格子

### 座標イメージ

A6 B6 C6 D6 E6 F6 G6

A3 B3 C3 D3 E3 F3 G3  
A2 B2 C2 D2 E2 F2 G2  
A1 B1 C1 D1 E1 F1 G1  

### 自陣地

- *You* 初期陣地：A1〜G1、A2〜G2
- *Opponent* 初期陣地：A6〜G6、A7〜G7
- *Creature* が移動したマスは、そのプレイヤーの陣地として扱う
  - 初期陣地は常に陣地として固定される

### マスの重複ルール

- *Creature* × *Creature*：不可
- *Creature* × 地雷：可
- 地雷 × 地雷：可

## 勝利条件

*Opponent* の初期陣地に *You* のコマが移動すること。

## ターン構成

1ターンは以下のフェイズで構成される。

1. 補充召喚フェイズ  
2. ドローフェイズ  
3. メインフェイズ  
4. エンドフェイズ  

### 1. 補充召喚フェイズ

- *Successors* を強制召喚する
- 補充はシステム処理として行われる
- *Creature* が死亡し、*Successor cost* で定められたターン数が経過した場合に補充される
- 召喚位置：自陣地内の任意のマス
- 置けない場合の処理：不明

### 2. ドローフェイズ

- 初期手札：3枚
- 手札上限：5枚
- 毎ターン必ずカードを1枚引く
- このフェイズ中はドロー以外の行動不可
- 上限超過時の処理：不明

#### 山札

- 山札に実体は存在しない
- 山札切れは発生しない
- カードの出現確率は W 値に依存する
- ターン経過による確率変動：要詳細定義（不明）

### 3. メインフェイズ

- 以下の行動を任意の順序で実行できる
  - カード使用
  - 移動権の行使

#### 行動制限

- 移動権：*You* につき *Creature* 1体を対象に、1ターン1回
- カード使用回数制限：なし

#### 移動と攻撃

- 攻撃はプレイヤーが宣言しない
- *Creature* が移動した先に敵 *Creature* が存在する場合、自動的に攻撃が発生する
- 攻撃は移動した *Creature* のみが行う

#### 戦闘解決

- 一方攻撃のみ
- 防御側の反撃は発生しない
- 攻撃側の攻撃力分、防御側の体力を減らす

### 4. エンドフェイズ

- 処理なし

## Creatures

- 各 *Creature* は以下のステータスを持つ
  - 体力（HP）
  - 攻撃力（AT）
  - *Successor cost*

### 移動

- 全 *Creatures* 共通
- 上下左右および斜めの8方向に1マス移動可能
- その場に留まることは不可

### スキル種別

- スキルなし
- パッシブスキル
- アクティブスキル（1回制限）

### Successor cost

- *Creature* は *Successor cost* を持つ
- *Successor cost* = 0 の場合、次の補充召喚フェイズで補充される

### Creature 一覧

表記：HP/AT｜successor cost = x

**スキルなし**
• Ameba（1/1｜x1）  
• Goblin（2/2｜x2）  
• Soldier（3/3｜x3）

**パッシブスキル**
• Lancer（2/1｜x3）  
　└ 範囲攻撃（前前）  
• Hobgoblin（1/1｜x3）  
　└ 死亡時：カードを1枚引く

**アクティブスキル（1回）**
• Ninja（2/2｜x4）  
　└ 追加で移動  
• Bomber（1/0｜x3）  
　└ 地雷カードを入手  
• Giant Boa（2/2｜x3）  
　└ 2マス前進  
• Alchemist（1/1｜x5）  
　└ カードを2枚引く

## カード一覧

表記：W = 出現確率の重さ（数字が大きいほど出現しにくい）

**移動系**
- Move [W1]  
　└ 1マス移動（攻撃なし）

- Assault [W5]  
　└ 突撃：1マス移動＋攻撃あり

**ダメージ系**
- Arrowrain [W1]  
　└ 敵モンスター1体に1ダメージ

- Rock Bombardment [W2]  
　└ 自陣地にいる敵モンスターに2ダメージ

- Lightning [W5]  
　└ 敵陣地にいるモンスターに3ダメージ

**補助・強化**
- Recharge [W3]  
　└ 自分のモンスター1体のスキルを再使用可能にする

- Doping [W3]  
　└ 自軍モンスター1体：+0/+1

- Barrier [W3]  
　└ 自軍モンスター1体：+1/+0

- Breath [W4]  
　└ 自軍モンスター1体：+1/+1

**トリッキー**
- Mine [W2]  
　└ 地雷を設置する（自分のモンスターの移動範囲内）

- Stealing [W4]  
　└ 相手の手札をランダムに1枚奪う
