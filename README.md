# ターン制クラロワ（仮）仕様書 v0.2

## ゲーム概要

ターン制の盤面上のモンスター移動とカード使用を組み合わせたボード×カードゲーム

- デッキ構築要素なし（共通山札方式）
- 思考スピードを要求しない
- 非公開情報は「手札」「補充」「地雷」
- テンポ重視
- カード1枚で読み合いが発生する設計を目指す
- 不利側が逆転できる要素を残す（地雷など）

### 用語

*You*: 自分
*Opponent*: 対戦相手
*Players*: *You*と*Opponent*の総称
*Creatures*: 盤面上に配置されるコマ
*Successores*: 補充される*Creature*
*Successor cost*: *Creature*死亡後、次の*Successor*が補充されるまでに必要なターン数

## 盤面仕様

- 盤面サイズ: 7*7
- 各マスは正方格子

```md
| A7 | B7 | C7 | D7 | E7 | F7 | G7 |
| A6 | B6 | C6 | D6 | E6 | F6 | G6 |
| A5 | B5 | C5 | D5 | E5 | F5 | G5 |
| A4 | B4 | C4 | D4 | E4 | F4 | G4 |
| A3 | B3 | C3 | D3 | E3 | F3 | G3 |
| A2 | B2 | C2 | D2 | E2 | F2 | G2 |
| A1 | B1 | C1 | D1 | E1 | F1 | G1 |
```

### 自陣地

- *You*初期陣地: A1-G1, A2-G2
- *Opponent*初期陣地: A6-G6, A7-G7
- *Creature*が移動した場所が陣地になる。
  - 初期陣地は陣地を変更できない。

### マスの重複ルール

- *Creature* * *Creature*: 不可
- *Creature* * 地雷: 可
- 地雷 * 地雷: 可

## ターン構成

1ターンは以下のフェイズで構成される。

1. 補充召喚フェイズ
2. ドローフェイズ
3. メインフェイズ
4. エンドフェイズ

### 1.補充召喚フェイズ

- *Successor*を強制召喚
- 補充はシステム処理
- 補充は*Creature*が死亡後に規定のターン数が経過した場合に発生
- 召喚位置: 自陣地内の任意の位置
- 置けない場合の処理：不明

### 2.ドローフェイズ

- 初期手札: 3枚
- 手札上限：5枚
- 毎ターン必ずカードを1枚引く
- 上限超過時の処理：不明

#### 山札

- 山札に実態はなく山札切れは発生しない
- カードの出現確率はレア度に依存する
  - ターンが進むごとにレアリティの高いカードが引きやすくなる（要詳細）

### 3.メインフェイズ

- 以下の行動を**任意の順序で実行可能**
  - カード使用
  - 移動権の行使
- 攻撃は移動後に権利があれば自動処理

#### 行動制限

- 移動権：*Players*につき*Creature*１体を対象に**1ターン1回**
- カード使用回数制限：なし

#### 攻撃

- 攻撃はプレイヤーが能動的に宣言しない
- **移動先に敵モンスターが存在する場合、攻撃権があれば自動的に攻撃が発生**
- 攻撃は移動後に移動した*Creature*のみ発生する

#### 戦闘解決

- 一方攻撃のみ
- 防御側の反撃は発生しない
- 攻撃側の*Creature*の攻撃力分、防御側の*Creature*の体力を減らす

### 4.エンドフェイズ

- 処理なし

## *Creatures*

- ステータスは体力と攻撃力と*Successor cost*

### 移動方向と距離

- 全*Creatures*共通で上下左右斜めの８方向に1マス移動
- その場に移動できない。

### スキル種別

- スキルなし*Creatures*
- パッシブスキル持ち*Creatures*
- アクティブスキル持ち*Creatures*（1回制限）

### *Successor cost*（補充フェーズの説明とかぶる？）

- *Creatures*は*Successor cost*を持つ
- *Successor cost*=0 なら次のターン補充フェイズに*Successores*が召喚される

### レア度（★）

- レア度はカードの強弱を示す指標
- 出現確率調整用の内部パラメータ
- 出現確率の式（不明）

　└ 2マス前進
• Alchemist（1/1｜x5）
　└ カードを2枚引く

## カード一覧

表記：W = 出現確率の重さ（数字が大きいほど出現しにくい）

**移動系**
- Move [W1]
　└ 1マス移動（攻撃なし）

- Assault [W5]
　└ 突撃：1マス移動＋攻撃あり

**ダメージ系**
- Arrowrain [W1]
　└ 敵モンスター1体に1ダメージ

- Rock Bombardment [W2]
　└ 自陣地にいる敵モンスターに2ダメージ

- Lightning [W5]
　└ 敵陣地にいるモンスターに3ダメージ

**補助・強化**
- Recharge [W3]
　└ 自分のモンスター1体のスキルを再使用可能にする

- Doping [W3]
　└ 自軍モンスター1体：+0/+1

- Barrier [W3]
　└ 自軍モンスター1体：+1/+0

- Breath [W4]
　└ 自軍モンスター1体：+1/+1

**トリッキー**
- Mine [W2]
　└ 地雷を設置する  
　　※ 自分のモンスターの移動範囲内のみ

- Stealing [W4]
　└ 相手の手札をランダムに1枚奪う
